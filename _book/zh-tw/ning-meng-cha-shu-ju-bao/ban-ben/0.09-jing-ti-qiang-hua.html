0.09 晶體强化

《檸檬茶0.04~0.08版本説明書》（滑鼠右鍵點擊這裏下載）

功能概述：

工作台

建築方塊：

  • 遠古遺骸（远古遗迹）
  • 方解石

裝飾方塊：

  • 鍾
  • 荷葉（浮萍）
  • 界伏盒（潜影盒）
  • 鐘乳石（滴水石锥）

晶體：

  • 紫水晶芽床（紫水晶母岩）
  • 燧石 ↔ 石英 ↔ 青金石 ↔ 紫水晶碎片 ↔ 紅石粉 ↔ 綠寶石 ↔ 鑽石
  • 海燈籠 ↔ 海磷晶體 ↔ 海磷碎片 ↔ 海磷方塊（海晶灯 ↔ 海晶沙砾 ↔ 海晶碎片 ↔ 海晶方块）
  • 雪球 ↔ 雪 ↔ 雪塊 ↔ 冰 ↔ 冰磚（浮冰） ↔ 藍冰

雜項：

  • 鱗甲
  • 熔岩桶 + 冰→ 岩漿块 + 鐵桶
  • 岩漿块 ↔ 岩漿球 → 烈焰粉 ↔ 烈焰桿
  • 烈焰粉 + 海磷晶體（海晶碎片） → 螢石粉 ↔ 熒光石

工具和戰鬥用品：

  • 三叉戟
  • 望遠鏡

工作台配方：

戰鬥用品

  • 海蹸晶體（海晶沙砾） + 海蹸碎片（海晶碎片） → 三叉戟

工具

  • (染色)玻璃片/紫水晶碎片 + 銅/鉄/金錠 → 望遠鏡

点击跳转

鐵桶

  • 冰 + 鐵桶 → 水桶
  • 雪塊 + 鐵桶 → 细雪桶
  • 岩漿块 + 鐵桶 ↔ 熔岩桶
  • 熔岩桶 + 冰→ 岩漿块 + 鐵桶

点击跳转

雜項

  • 海龜蛋 → 鱗甲
  • 烈焰粉 → 烈焰桿（烈焰棒）
  • 海磷晶體（海晶碎片） + 烈焰粉 → 螢石粉
  • 熒光石（荧石） → 螢石粉

点击跳转

水形態

  • 雪球 ↔ 雪 ↔ 雪塊 ↔ 冰 ↔ 冰磚（浮冰） ↔ 藍冰

紫水晶

  • 紫水晶碎片 ↔ 紫水晶方塊 ↔ 紫水晶芽床（紫水晶母岩）

寶石

  • 燧石 ↔ 石英 ↔ 青金石 ↔ 紫水晶碎片 ↔ 紅石粉 ↔ 綠寶石 ↔ 鑽石

玻璃

  • (染色)玻璃片 → 玻璃
  • (染色)玻璃 + 染料 → 染色玻璃
  • (染色)玻璃片 + 染料 → 染色玻璃片
  • 染色玻璃 → 玻璃
  • 染色玻璃片 → 玻璃片

石英

  • 石英方塊/(平滑)石英半磚 → 地獄石英（下界石英）
  • (平滑)石英方塊/浮雕石英方塊/石英磚/石英柱/(平滑)石英階梯 → 石英方塊

海磷（海晶）

  • 海燈籠（海晶灯） ↔ 海磷晶體 ↔ 海磷碎片
  • 海磷碎片 ↔ 海磷石/海磷石階梯/海磷石牆/海磷石半磚
  • 海磷碎片 ↔ 暗海磷石/暗海磷石階梯/暗海磷石半磚/暗海磷石階梯
  • 海磷碎片 ↔ 海磷石磚/海磷石磚階梯/海磷石磚半磚

点击跳转

建築方塊

  • 骨塊 + 鐘乳石方塊 → 遠古遺骸
  • 海磷碎片 + 鐘乳石 → 方解石

点击跳转

裝飾方塊

  • 荷葉 + 骨粉 → 荷葉（浮萍）
  • 木棍 + 金錠 → 鍾
  • 爆開的哥萊果（爆裂紫颂果） + 木桶 → 界伏盒（潜影盒）
  • 鐘乳石方塊（滴水石块） → 鐘乳石（滴水石锥）

点击跳转

合成：釀造

  • 岩漿块 → 岩漿球（岩浆膏）

点击跳转

燒製配方：

高爐

  • 紫水晶碎片/玻璃瓶 → 玻璃
  • 腐肉 → 皮革
  • 岩漿球（岩浆膏） → 烈焰粉

点击跳转

熔爐

  • 腐肉 → 兔子皮
  • 岩漿球（岩浆膏） → 烈焰粉

点击跳转

鍛造台配方

升級：防具

  • 海龜殼 + 金錠 → 黃金頭盔

点击跳转
